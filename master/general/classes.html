類別

類別是什麼？

類別就是普通的 PHP 類別。它不需要擴展任何東西，或跟隨任何協定除了命名協定，Fuel 內的所有其他類別也相同。

class Session

將載入 app/classes/session.php。

載入類別

不像一些其他框架，類別不需要手動載入。當你在代碼中引用它們（控制器、模型等），它們將會自動載入。

子目錄中的類別

就像 Controllers，類別必須是小寫且第一個字母大寫，同時底線將置類別於子目錄。

Class Session_Driver

將載入 app/classes/session/driver.php

初始化你的類別

類別被載入時，使 Fuel 的自動載入器實施某些自動的任務是可能的，有點像類別的實例 __construct() 所做的。 你可以通過添加一個公開的靜態 _init() 方法到類別達成。

class Example {
	public static function _init()
	{
		// 緊接在載入類別後被呼叫
	}
}

如果一個被載入的類別有它自己的 _init() 方法及父層，必須跟著被載入，父層的初始化方法不會被呼叫，除非你使用 parent::_init();
