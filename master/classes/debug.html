Debug 類別

The Debug class is a simple utility for debugging variables, objects, arrays, etc by outputting information to the display. This is not meant as a one-stop-shop to all of your debugging needs, but as a simple way to throw out information quickly to check things as you go.

dump(*$args)

The dump method outputs multiple mixed values to the browser in a formatted structured way and includes backtrace information so you know where it is coming from.

靜態 是
參數 Unlimited mixed arguments
回傳 空
範例 
Debug::dump($stuff, 'junk', array('whatever'));

backtrace()

The backtrace method shows the file and line the method is called from and a list of all previous.

靜態 是
參數 無
回傳 空
範例 
Debug::backtrace();
