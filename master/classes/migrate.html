Migrate 類別

Migrate 類別能讓你從你的控制器運行、穿行與恢復 遷移 。

current()

運行 up 與 down 方法到遷移配置裡當前設定的遷移。

靜態 是
參數 無
回傳 布林
範例 
Migrate::current();

latest()

為曾經的遷移運行 up 方法，在遷移文件夾裡當前與最後的版本之間。

靜態 是
參數 無
回傳 布林
範例 
Migrate::latest();

version($version)

version 方法將移動遷移往上或往下，以設定架構在一個特定的遷移。

靜態 是
參數
變數 預設 描述
$version 必要 你要遷移的版本。
回傳 混合 - 依據 $return_type。
範例 
Migrate::version($version)
