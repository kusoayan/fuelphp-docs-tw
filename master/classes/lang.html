Lang 類別

Lang 類別能讓你在你的應用程式中，使用語言檔案設定語言變數。

load($file, $group = null)

load 方法能讓你載入一個語言檔案。

靜態 是
參數
變數 預設 描述
$file 必要 所需的文件路徑。你可以加上一個命名空間的前綴以載入一個已載入的套件或模組的配置檔案。
$group 
null
 設定要使用的語言組。
回傳 空
範例 
// 語言檔案的例子：
return array(
	'hello' => 'Hello :name',
	'something'=> 'something :name!',
	'test'=> array('hello' => 'Hello', 'something' => 'Plop') // Group
);

// 載入 example.php。
// 註：如果在配置裡沒有語言被設定，將會退至 English。
Lang::load('example');

// 將載入給定的檔案進 'test' 群組。
Lang::load('example', 'test');

// 輸出 Plop
$this->output = Lang::line('test.test.something');

// 將從模組 'foo' 載入 example 語言檔案進 'bar' 群組。
Lang:load('foo::example', 'bar');

get($line, $params = array())

get 方法能讓你從語言檔案取得特定行。

靜態 是
參數
變數 預設 描述
$line 必要 所需行的識別符。
$parameters 
array()
 設定一個能被傳遞至行的參數陣列。
回傳 混合。如果找到是字串，否則是 false
範例 
// 輸出 Hello world
$this->output = Lang::get('hello', array('name' => 'world');

// 輸出 Plop
$this->output = Lang::get('test.something');

set($line, $value, $group = null)

set 方法能讓你設定一個特定行至語言檔案。

靜態 是
參數
變數 預設 描述
$line 必要 所需行的識別符。
$value 
array()
 設定行的值。
$group 
null
 設定要使用的語言組。
回傳 空
範例 
// 回傳 true
Lang::set('hello', 'Ahoy!');

// 使用群組
Lang:set('hello', 'Ahoy!', 'test');

// 也可如上運作
Lang::load('test', 'test);
Lang::set('hello', 'Ahoy!');

程序助手

__($string, $params = array())

__ 函數是 Lang::line 的別名。

參數
變數 預設 描述
$string 必要 所需行的識別符。
$parameters 
array()
 設定一個能被傳遞至行的參數陣列。
回傳 字串，來自 Lang::line 的結果
範例 
// 輸出 Hello world
$this->output = __('hello', array('name' => 'world');

// 輸出 Plop
$this->output = __('test.something');
