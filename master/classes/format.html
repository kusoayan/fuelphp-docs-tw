Format 類別

Format 類別幫助你在各種格式間轉換，如 XML、JSON、CSV 等。

factory($data = null, $from_type = null)

factory 方法回傳一個新格式物件。

靜態 是
參數
變數 預設 描述
$data 
null
 要轉換的資料。
$from_type 
null
 所提供的資料格式。
回傳 Fuel\Core\Format 物件
範例 
$array = array('foo' => 'bar');
print_r( Format::forge( $array ) );

// 回傳
Fuel\Core\Format Object
(
	[_data:protected] => Array
		(
			[foo] => bar
		)
)

to_array($data = null)

to_array 方法回傳給定的資料為陣列。不要直接呼叫，使用 factory 方法如同範例所描述。

靜態 否
參數
變數 預設 描述
$data 
null
 要轉換的資料。
回傳 陣列
範例 
$json_string = '{"foo":"bar","baz":"qux"}';
print_r( Format::forge( $json_string, 'json' )->to_array() );

// 回傳
Array
(
    [foo] => bar
    [baz] => qux
)

to_xml($data = null, $structure = null, $basenode = 'xml')

to_xml 方法回傳給定的資料為一個 XML 字串。不要直接呼叫，使用 factory 方法如同範例所描述。

靜態 否
參數
變數 預設 描述
$data 
null
 要轉換的資料。
$structure 
null
 SimpleXMLElement 類別物件。
$basenode "xml" XML 標記的基本節點。
回傳 字串
範例 
$array = array('foo' => 'bar', 'baz' => 'qux');
print_r( Format::forge( $array )->to_xml() );

// 回傳
// <?xml version="1.0" encoding="utf-8"?>
// <xml>
// 	<foo>bar</foo>
// 	<baz>qux</baz>
// </xml>

to_csv()

to_csv 方法回傳給定的資料為一個 CSV 字串。不要直接呼叫，使用 factory 方法如同範例所描述。

靜態 否
回傳 字串
範例 
$json_string = '{"foo":"bar","baz":"qux"}';
print_r( Format::forge( $json_string, 'json' )->to_csv() );

// 回傳
//
// foo,baz
// "bar","qux"

to_json()

to_json 方法回傳給定的資料為一個 JSON 字串。不要直接呼叫，使用 factory 方法如同範例所描述。

靜態 否
回傳 字串
範例 
$array = array('foo' => 'bar', 'baz' => 'qux');
print_r( Format::forge( $array )->to_json() );

// 回傳
// {"foo":"bar","baz":"qux"}

to_serialized()

to_serialized 方法回傳給定的資料為一個序列化字串。不要直接呼叫，使用 factory 方法如同範例所描述。

靜態 否
回傳 字串
範例 
$array = array('foo' => 'bar', 'baz' => 'qux');
print_r( Format::forge( $array )->to_serialized() );

// 回傳
// a:2:{s:3:"foo";s:3:"bar";s:3:"baz";s:3:"qux";}

to_php()

to_php 方法回傳給定的資料為 PHP 表達字串中的資料。你可以傳進 eval() 或用做其他瘋狂的事情。

靜態 否
回傳 字串
範例 
$array = array (1, 2, array ("a", "b", "c"));
echo Format::forge( $array )->to_php();

// 回傳
"array (
  0 => 1,
  1 => 2,
  2 =>
  array (
    0 => 'a',
    1 => 'b',
    2 => 'c',
  ),
)"
