Event 類別

Event 類別能讓你在無須修改任何核心檔案之下與 Fuel 核心互動。

register($event, $callback)

register 方法允許檔案註冊一個物件，將在 trigger 方法被呼叫時運行。

靜態 是
參數
變數 預設 描述
$event 必要 要註冊你的代碼的事件。
$callback 必要 回調方法。
回傳 布林
範例 
Event::register('user_login', 'Class::method');

trigger($event, $data = '', $return_type = 'string')

trigger 方法用於觸發或啟動通過 register 方法的回調。

靜態 是
參數
變數 預設 描述
$event 必要 你要觸發的事件。
$data 
''
 任何你想要送到已註冊方法的資料。
$return_type 
'string'
 你期望回傳的資料類型。（字串 (string)、陣列 (array)、json、無 (none)、序列化 (serialized)） - 預設是字串
回傳 混合 - 依據 $return_type。
範例 
Event::trigger('user_login', $data)

has_events($event)

has_events 方法是可利用的，所以你可以檢查特殊的已註冊事件是否有觸發器。

靜態 是
參數
變數 預設 描述
$event 必要 你要檢查的事件。
回傳 布林
範例 
Event::has_events('user_login')
