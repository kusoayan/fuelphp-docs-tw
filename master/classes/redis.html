Redis 類別

Redis 類別能讓你與 Redis 鍵 - 值儲藏相互作用。

instance($name = 'default')

建立一個新的已命名的 redis 的實例，或回傳一個已命名的實例如果它先前已建立。

靜態 是
參數
變數 預設 描述
$name 
'default'
 請求的 Redis 實例名稱，如應用程式配置 db.php 檔案所定義。
回傳 Fuel\Core\Redis 物件
範例 
// 建立 Redis 'mystore' 實例
$redis = Redis::instance('mystore');

Redis 命令

一旦你使用 instance() 方法建立一個 Redis 物件，每個有效的 Redis 命令（見 Redis 文件）作為該物件的方法。

一些範例：

// 建立 Redis 'mystore' 實例
$redis = Redis::instance('mystore');

// 建立一些測試資料
$redis->rpush('particles', 'proton');
$redis->rpush('particles', 'electron');
$redis->rpush('particles', 'neutron');

// 提取範圍
$particles = $redis->lrange('particles', 0, -1);

// 計算元素的數量
$particle_count = $redis->llen('particles');

// 顯示結果
echo "<p>The {$particle_count} particles that make up atoms are:</p>";
echo "<ul>";
foreach ($particles as $particle) {
  echo "<li>{$particle}</li>";
}
echo "</ul>";
