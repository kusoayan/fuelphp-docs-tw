需求

網頁伺服器
  • 任何網頁伺服器。Fuel 已在 Apache、IIS 與 Nginx 測試。
  • 運行於所有主要的作業系統包含 *nix 與 Windows。
PHP版本
  • 5.3或更高的版本是必要的。
PHP 擴充

Fuel 使用 PHP 定義為「擴展」的功能。為了避免任何可能的相容性問題，Fuel 只使用被撰寫在 PHP 網站文件中作為 PHP 代碼一部份的擴展。如果使用 PECL 庫中的部分擴展（這需要由系統管理員手動安裝），如果存在的話，Fuel 將會使用這些擴展，否則將修正。

以下是使用的擴展列表，以及哪些類別（classes）使用到：

擴展 用於 評論
fileinfo Upload 文件
請注意，在 Windows 平台上，可能需要手動安裝一個 DLL。
mbstring n/a 文件
如果不存在，該框架將沒有多字節支援。你可以確認 MBSTRING 常數以了解是否你的安裝已啟用多字節支援。
mcrypt Crypt 文件
如果不存在，PHPSecLib 會被用來模擬它的功能。

推薦

  • 網頁伺服器 - 運行在 *nix 上的 Apache。
  • PHP 版本 - 5.3.2
  • mod_rewrite - 從 URL 中移除 index.php。
  • PHPUnit - 3.5.12
