Oil 工具

Oil 工具是特別的套件，命令可被使用於許多地方，以促進快速發展，幫助測試你的應用程式及運行任務。

Oil 是一個可選的命令列實用工具，旨在幫助加快發展，提供多種函數：

  • 產生 - 建構 MVC 模組、遷移及整個腳手架。
  • 精煉 - 運行任務如遷移與你自定義的。
  • 套件 - 安裝、升級及移除套件。
  • 控制台 - 使用 shell 即時測試你的代碼。

這些命令都以不同的方式來實現不同的東西，但進入前，最好確認你是在正確的文件夾，且 Oil 是可運行的。 

$ cd Sites/fuel
$ php oil -v
Fuel: 1.1

行內幫助

Oil 含有自己的基本文件，可以通過鍵入命令找到：

$ php oil help

Usage:
  php oil [console|generate|help|test|package]

Runtime options:
  -f, [--force]    # Overwrite files that already exist
  -s, [--skip]     # Skip files that already exist
  -q, [--quiet]    # Suppress status output

Description:
  The 'oil' command can be used in several ways to facilitate quick development, help with
  testing your application and for running Tasks.

Documentation:
  http://docs.fuelphp.com/packages/oil/intro.html

這些命令都有自己的幫助章節，可以通過鍵入找到：

$ php oil generate help
