下載 Fuel

Fuel 現在是 v1.2 版，你可以使用以下連結下載：

  • 發行版
    • 下載 1.2
    • 下載 1.1
    • 下載 1.0.1
    • 下載 1.0
  • 儲存庫
    • Github 上的 Fuel
    • Github 上的 Fuel Core
    • Github 上的 Docs
    • Github 上的 Fuel Orm
    • Github 上的 Fuel Auth
    • Github 上的 Fuel Parser
    • Github 上的 Fuel Oil
    • Github 上的 Fuel Email
  • 使用者貢獻的 Fuel 套件
    • Github 上的 Fuel-packages 帳戶
      發送一個訊息到此帳戶或到 Twitter 上的 @fuelphp， 讓你的套件 fork 進 fuel-packages。

從命令列安裝 Fuel 認可的套件，使用 Oil 實用工具。

貢獻核心

Fuel 透過社群的方式開發，而且我們樂見任何人的參與。如果你有興趣，請花點時間閱讀我們的 編碼標準。 任何不符合這些準則的貢獻將不被接受。接著請：fork、commit 與 pull-request，如果你不知道如何做： http://help.github.com/forking/

貢獻套件

你有一些其他 Fuel 使用者可能感興趣的東西？那就建立一個 套件。 如果它撰寫得當，並符合我們的 撰碼標準，我們將在此網頁上發佈， 並讓它可通過 oil 從命令列安裝。
